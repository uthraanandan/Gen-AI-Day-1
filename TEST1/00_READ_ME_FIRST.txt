## 🏗️ Architecture At A Glance

```
User (http://localhost:3000)
    │
    ├─ Input: "Ask me anything..."
    │
    ▼
React UI (beautiful, responsive)
    │
    ├─ POST /ask
    │
    ▼
Express Server (http://localhost:5000)
    │
    ├─ Step 1: GPT Decision
    │   └─ "Needs search?" → YES/NO
    │
    ├─ Step 2: Web Search (if YES)
    │   └─ Tavily API → 5 results
    │
    ├─ Step 3: Answer Synthesis
    │   └─ GPT + search results → answer
    │
    ▼
Response JSON
    {
      answer: "...",
      sources: [...],
      usedSearch: true/false
    }
    │
    ▼
React UI Display
    └─ Answer + clickable sources
```

## 🆘 Troubleshooting

Common issues and solutions:

```

Fix: Kill process or use different port

Issue: API key errors
Fix: Check .env file has correct keys

Issue: Backend won't start
Fix: Run npm install first

Issue: Frontend can't connect
Fix: Ensure backend running on 5000

Issue: No search happening
Fix: Verify Tavily API key valid
```
